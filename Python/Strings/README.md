## Strings

### Prerequisites
Understand **substrings** and **indexing of strings**.

---

### Problem 1: Longest Sequence of Vowels
Given a string, find the **maximum length of vowels occurring in sequence** and their **indexes**.

**Example:**

```
Input: "madnesaeiokrj"
Output:
aeiou
6,10
```
